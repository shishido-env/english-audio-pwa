**Goal:** `useLibrary` を Server Actions ベースの DB 同期に置換し、localStorage を読み取り高速化用キャッシュ層へ降格する。サインインしたユーザーは異なる端末からも同じデッキが見える状態になる。

**Tech Stack:** Next.js 15 Server Actions (`"use server"`), Drizzle ORM (Phase F5 で導入済み), Zod (新規), `revalidatePath`, Vitest + `vi.mock` でアクションをスタブ

## 設計判断の前提

### 1. UI 型と DB 型のブリッジ

DB は `decks` と `cards` を 2 テーブルに分けて保存します。UI は `Deck { id, name, pairs: Pair[], importedAt }` を期待します。F6 では **Server Action 内で JOIN して UI 形へ整形する** 方針を採用し、UI 型 (`src/types.ts`) は基本変更しません。

| 場所 | 形 |
|---|---|
| DB (`src/db/schema.ts`) | `Deck` (no pairs), `Card` (deckId + ja + en + position) |
| Server Action 戻り値 | UI 形の `Library { decks: Deck[], activeId: null }` |
| クライアント (`src/types.ts`) | 既存 `Library { decks: Deck[], activeId: string \| null }` |

`activeId` は Server Action 戻り値では常に `null`。`useLibrary` 側で localStorage の `activeId` を merge して最終表示する。

### 2. 認可

すべての Server Action 冒頭で `const { userId } = await auth();` を呼び、`userId` が無ければ `{ ok: false, error: "unauthorized" }` を返す。各 SQL 文の `where` 句に `eq(decks.userId, userId)` を必ず含めて他人のデータを返さない。

### 3. 入力検証

Zod スキーマで型検証。エラー時は `{ ok: false, error: "invalid_input" }` を返す。

### 4. 一括 INSERT のトランザクション

`createDeckFromCsv` は 1 つの `decks` 行 + N 個の `cards` 行を一貫性をもって作る必要があるため `db.transaction()` を使う。Neon `Pool` ドライバはトランザクションをサポート。

### 5. テスト戦略

実 DB なしでテストするため、`vi.mock("@/db/client", ...)` で `getDb()` を返すモックを作る。`@clerk/nextjs/server` の `auth()` も同様にモック。各テストはアクションを直接 import + 呼出。

Server Action 自体は `"use server"` ディレクティブ付きの async 関数で、Next.js コンパイラがそれを RPC 化するが、Vitest 環境では普通の関数として動く。

### 6. キャッシュ無効化

mutation 系の Server Action（create/rename/remove/clear）は `revalidatePath('/decks')` を呼ぶ。`useLibrary` は呼出後に `getLibrary()` を再実行するので二重整合性が確保される。

## 前提知識（実装者向け）

### Server Action の基本形

`src/actions/library.ts` の先頭に `"use server"` を書くと、その file から export される async 関数すべてが Server Action になる。Client Component から `import` して直接 await できる（Next.js が RPC に変換する）。

```ts
"use server";
import { auth } from "@clerk/nextjs/server";

export async function getLibrary(): Promise<Result<Library>> {
  const { userId } = await auth();
  if (!userId) return { ok: false, error: "unauthorized" };
  // ...
  return { ok: true, data: { decks: [...], activeId: null } };
}
```

### Vitest でのモック方法

```ts
import { vi } from "vitest";

vi.mock("@clerk/nextjs/server", () => ({
  auth: vi.fn().mockResolvedValue({ userId: "user_test" }),
}));

vi.mock("@/db/client", () => ({
  getDb: vi.fn(),
}));

// テスト内で `getDb` の戻り値を再構成して特定の SQL チェーンを返させる
```

実装者が知っておくべき: Drizzle のクエリチェーンは fluent builder で、`db.select().from(...).where(...)` が最終的に Promise を返す。モックでは `select` が `from` を返し、`from` が `where` を返し、最後の `.where` または `.all()` のような確定メソッドで Promise を返す形を作る。

このプロジェクトでは複雑なチェーンを避けるため、テストでは可能な限り `getDb()` が返す `db` を **dummy object でなく単純な関数モック** にして、アクション側のロジックだけを検証する（DB 呼出の SQL 構造ではなく、入力 → モック呼出引数 → 出力の流れを検証）。

### Server Action と revalidatePath

mutation 後に `import { revalidatePath } from "next/cache"; revalidatePath("/decks");` を呼ぶ。テスト時は `revalidatePath` もモックする（呼出引数だけ検証）。

### localStorage キャッシュ層の役割

| 操作 | localStorage の動き |
|---|---|
| アプリ起動 | `loadLibrary()` でキャッシュ即読 → useState 初期値 |
| `getLibrary()` 完了 | 戻り値を `saveLibrary` でキャッシュ更新 |
| mutation 成功 | サーバ最新を再フェッチ → `saveLibrary` でキャッシュ更新 |
| サインアウト | F7 で扱う（このフェーズではスコープ外） |

Create `src/actions/library.ts` with this exact content:

```ts
"use server";

import { auth } from "@clerk/nextjs/server";
import { eq } from "drizzle-orm";
import { getDb } from "@/db/client";
import { decks } from "@/db/schema";
import type { Library, Pair } from "@/types";

export type Result<T> =
  | { ok: true; data: T }
  | { ok: false; error: string };

export async function getLibrary(): Promise<Result<Library>> {
  const { userId } = await auth();
  if (!userId) return { ok: false, error: "unauthorized" };

  const db = getDb();
  const rows = await db.query.decks.findMany({
    where: eq(decks.userId, userId),
    with: { cards: true },
    orderBy: (decks, { desc }) => [desc(decks.importedAt)],
  });

  const uiDecks = rows.map((row) => {
    const sortedCards = [...row.cards].sort(
      (a, b) => a.position - b.position,
    );
    const pairs: Pair[] = sortedCards.map((c) => ({ ja: c.ja, en: c.en }));
    return {
      id: row.id,
      name: row.name,
      pairs,
      importedAt:
        row.importedAt instanceof Date
          ? row.importedAt.getTime()
          : Number(row.importedAt),
    };
  });

  return {
    ok: true,
    data: { decks: uiDecks, activeId: null },
  };
}
```

import { z } from "zod";
import { revalidatePath } from "next/cache";
import { cards } from "@/db/schema";

const createDeckSchema = z.object({
  name: z.string().min(1).transform((s) => s.trim()).refine((s) => s.length > 0),
  pairs: z
    .array(z.object({ ja: z.string(), en: z.string() }))
    .min(1),
});

export async function createDeckFromCsv(input: {
  name: string;
  pairs: Pair[];
}): Promise<Result<{ id: string }>> {
  const { userId } = await auth();
  if (!userId) return { ok: false, error: "unauthorized" };

  const parsed = createDeckSchema.safeParse(input);
  if (!parsed.success) return { ok: false, error: "invalid_input" };

  const db = getDb();
  const importedAt = new Date();

  const result = await db.transaction(async (tx) => {
    const inserted = await tx
      .insert(decks)
      .values({
        userId,
        name: parsed.data.name,
        importedAt,
      })
      .returning();
    const deck = inserted[0];
    if (!deck) throw new Error("insert_failed");
    const cardRows = parsed.data.pairs.map((p, i) => ({
      deckId: deck.id,
      ja: p.ja,
      en: p.en,
      position: i,
    }));
    await tx.insert(cards).values(cardRows).returning();
    return { id: deck.id };
  });

  revalidatePath("/decks");
  return { ok: true, data: result };
}
```

"use client";

import { useCallback, useEffect, useState } from "react";
import type { Deck, Library } from "@/types";
import { parseCsv } from "@/lib/csv";
import {
  loadLibrary,
  saveLibrary,
  loadActiveId,
  saveActiveId,
} from "@/lib/storage";
import {
  getLibrary,
  createDeckFromCsv,
  renameDeck as renameDeckAction,
  removeDeck as removeDeckAction,
  clearAllDecks,
} from "@/actions/library";

const EMPTY_LIBRARY: Library = { decks: [], activeId: null };

function stripExtension(filename: string): string {
  return filename.replace(/\.[^.]+$/, "");
}

function mergeActiveId(library: Library, fallback: string | null): Library {
  if (library.activeId) return library;
  if (!fallback) {
    return { ...library, activeId: library.decks[0]?.id ?? null };
  }
  const exists = library.decks.some((d) => d.id === fallback);
  return { ...library, activeId: exists ? fallback : library.decks[0]?.id ?? null };
}

export function useLibrary() {
  const [library, setLibrary] = useState<Library>(() => {
    if (typeof window === "undefined") return EMPTY_LIBRARY;
    const cached = loadLibrary();
    const activeId = loadActiveId();
    return mergeActiveId(cached, activeId);
  });
  const [isLoading, setIsLoading] = useState(true);

  const refetch = useCallback(async () => {
    const res = await getLibrary();
    if (!res.ok) {
      // 認証切れやサーバエラー時はキャッシュを保持
      setIsLoading(false);
      return;
    }
    const activeId = typeof window !== "undefined" ? loadActiveId() : null;
    const merged = mergeActiveId(res.data, activeId);
    setLibrary(merged);
    if (typeof window !== "undefined") {
      saveLibrary({ ...merged, activeId: null });
    }
    setIsLoading(false);
  }, []);

  useEffect(() => {
    void refetch();
  }, [refetch]);

  const activeDeck: Deck | null =
    library.decks.find((d) => d.id === library.activeId) ?? null;

  const importCsv = useCallback(
    async (filename: string, content: string) => {
      const pairs = parseCsv(content);
      if (pairs.length === 0) throw new Error("有効な行がありません");
      const name = stripExtension(filename);
      const res = await createDeckFromCsv({ name, pairs });
      if (!res.ok) throw new Error(res.error);
      await refetch();
      // 新作成デッキを active に
      setLibrary((lib) => {
        const exists = lib.decks.some((d) => d.id === res.data.id);
        if (!exists) return lib;
        if (typeof window !== "undefined") saveActiveId(res.data.id);
        return { ...lib, activeId: res.data.id };
      });
    },
    [refetch],
  );

  const selectDeck = useCallback((id: string) => {
    setLibrary((lib) => {
      if (!lib.decks.some((d) => d.id === id)) return lib;
      if (typeof window !== "undefined") saveActiveId(id);
      return { ...lib, activeId: id };
    });
  }, []);

  const renameDeck = useCallback(
    async (id: string, name: string) => {
      const trimmed = name.trim();
      if (trimmed === "") return;
      const res = await renameDeckAction({ id, name });
      if (!res.ok) return;
      await refetch();
    },
    [refetch],
  );

  const removeDeck = useCallback(
    async (id: string) => {
      const res = await removeDeckAction({ id });
      if (!res.ok) return;
      const wasActive = library.activeId === id;
      await refetch();
      if (wasActive) {
        // refetch 後の最初のデッキを active に（mergeActiveId が拾う）
        setLibrary((lib) => {
          const next = lib.decks[0]?.id ?? null;
          if (typeof window !== "undefined") saveActiveId(next);
          return { ...lib, activeId: next };
        });
      }
    },
    [library.activeId, refetch],
  );

  const clearAll = useCallback(async () => {
    const res = await clearAllDecks();
    if (!res.ok) return;
    if (typeof window !== "undefined") saveActiveId(null);
    await refetch();
  }, [refetch]);

  return {
    library,
    activeDeck,
    isLoading,
    importCsv,
    selectDeck,
    renameDeck,
    removeDeck,
    clearAll,
  };
}
```
